# Pokemon GraphQL API

> A high-performance Pokemon TCG GraphQL API with cursor pagination, dataloader batching, and interactive schema exploration.

- **Apollo Server 4** — Production-grade GraphQL runtime on Bun
- **Cursor Pagination** — Relay-style connections for cards and sets
- **Dataloader Batching** — Request-scoped N+1 query elimination
- **Interactive IDE** — Built-in GraphiQL with example queries
